# 🎬 Boost for Letterboxd

A Chrome extension that puts Letterboxd front and center in your Google searches.

![Promo](assets/promo_large_1400x560.png)

## Features

**Letterboxd Banner** — When you search for a movie, director or actor on Google, a direct link to Letterboxd appears at the top of the results.

**Boosted Results** — Existing Letterboxd results in the Google page are automatically moved to the top with a visual highlight.

**Right-Click Search** — Highlight any text on any website, right-click, and choose "Search on Letterboxd" to jump straight there.

**Smart Detection** — The extension detects cinema-related searches using keywords in French and English (film, actor, Netflix, review, streaming, etc.).

## Installation

### Chrome Web Store (recommended)

Install directly from the [Chrome Web Store](https://chromewebstore.google.com/detail/boost-for-letterboxd/jfnkjbammnogkcfpflgobdjadmcbjigd).

### Manual install (dev mode)

1. Clone this repo
   ```bash
   git clone https://github.com/AlexherDev69/boost-for-letterboxd.git
   ```
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the cloned folder
